Engineering Team Minutes System

A meeting-minutes & action/risk tracking system in Excel + VBA. Built solo for a local government engineering team. Deployed inside a locked-down SharePoint environment — no plugins, no admin rights.

By Adina Lieblich · Civil engineer · Perth, Australia · AI-assisted (Claude)

Get the kit → v1.0 release (.zip) Field guide PDF (40 pp) Repo

Two things in this repo. The system — VBA modules, Python verification scripts, sanitised example workbook scaffolding, style guide. And the field guide — what AI gets wrong when you ask it to write corporate-grade VBA, and what to do about it.

Start with What AI Won't Tell You if you want the lessons. Start with How To Use if you want to understand how the system runs.

If you want to run this at your team

Pull down the v1.0 kit. Read How To Use. Read the field guide for the AI patterns. The VBA modules in src/vba/ are the actual build — feed them into a fresh Excel workbook macro module and run them in order. The system was built for one engineering team but the patterns transfer.

Bugs, questions, "this didn't work for me" — open an issue on the repo.
